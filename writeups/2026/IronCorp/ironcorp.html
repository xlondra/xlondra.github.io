Lab: Iron Corp

  • Date: 2026-03-17
  • Platform: TryHackMe
  • Difficulty: Hard
  • Link: Open Lab
  • Tags: smb windows metasploit

Reconnaissance

Lets first start scanning the target network with nmap

└─$ sudo nmap -p- ironcorp.me -Pn

This shows us 7 open ports.
Next, the version scan, we want to know what exactly is running on the ports 

└─$ sudo nmap -p53,135,3389,8080,11025,49668,49670 -sV ironcorp.me -Pn

This result strongly indicates that this service is running on a Windows Server (IIS)
We can also see 2 webservers running. One on port 8080 and one on port 11025

Exploitation

The SMB service was vulnerable to EternalBlue which allowed remote code execution through the Metasploit module.
