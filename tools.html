Tools & Resources

GTFOBins

Quick reference for abusing Linux binaries during privilege escalation. Essential when dealing with SUIDs and sudo misconfigurations.

linux privesc

HackTricks

Massive knowledge base covering enumeration, exploitation, and privilege escalation across Linux, Windows, and web.

cheatsheet general

Revshell Generator

Generate reverse shell payloads quickly for different environments. Useful during exploitation when you need a working shell fast.

exploit shell

CyberChef

Powerful data manipulation tool for decoding, encoding, hashing, and analysis. Extremely useful for handling payloads and transforming data.

encoding analysis

Base64 Guru

Simple tool for encoding/decoding base64. Useful when dealing with payloads, tokens, or obfuscated data during testing.

encoding utility

Censys

Search engine for internet-exposed hosts and services. Less commercial than Shodan, equally powerful for passive recon.

recon osint

Exploit-DB

Database of public exploits and vulnerabilities. Useful for finding proof-of-concept exploits during enumeration and exploitation phases.

exploit database

ANY.RUN

Interactive online sandbox for malware analysis. Lets you detonate samples in a safe environment and observe behavior in real time.

malware sandbox

VirusTotal

Aggregate file and URL scanning across 70+ antivirus engines. Good first step when triaging a suspicious sample.

malware triage
